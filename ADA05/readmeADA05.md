
Este proyecto forma parte de una actividad colaborativa cuyo objetivo principal fue practicar el uso de herramientas de control de versiones, específicamente GitHub. Cada equipo debía implementar uno de los algoritmos de ordenamiento asignados y posteriormente realizar un pull request hacia el repositorio base del profesor. Como producto final, se obtuvo un único programa en C que integra los 10 algoritmos de ordenamiento trabajados por todos los equipos.
Los algotimos abordados fueron:
-Bubble Sort 
-Selection Sort
-Insertion Sort
-Quick Sort
-Merge Sort
-Cocktail Sort
-Heap Sort 
-Tree Sort 
-Tournament Sort
-Shell Sort 
El programa permite crear arreglos dinámicos de números flotantes —ya sea de forma aleatoria o ingresados por el usuario— y aplicar distintos métodos de ordenamiento a elección. También incorpora utilidades como copias del arreglo, conteo de operaciones en algunos algoritmos y representaciones auxiliares para estructuras como árboles binarios.


Compilación: (desde terminal)
gcc sorting.c -o program.exe
Ejecución:
./program.exe
